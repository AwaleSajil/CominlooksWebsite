Chapter-3 Creating Ordered And Unordered List

If you have to arrange large numberof items to make a group then you can mae listof items. In HTML it provides 3 types of lists: Defination List, Ordered List,Unordered List.

Defination Lists

It is not commonly used lists. It is mainly used to give defination of a term. The <DL> is used to define this lists and <Dt> and <DD> tags are used to define defination term and defination respectively.

Unordered Lists

An unordered lists mae ulleted list of items. The <UL> defines unordered lists. The <LI> subtag defines items in the list. Type is only one attribue of this tag which takes values like disk/circle/square.

Ordered Lists

An ordered list is similar to unordered list but it lists items using numbers or characters. The <OL> defines this tag and <LI> subtag defines items in the list. Type and start are its two attributes and it takes the values like 1/A/a/i and N(numbers) respectively.

Chapter-4 Inserting Image And Hyperlink

Inserting Image

The <IMG> tag defines images in HTML, it has no ending tag and has following attributes:

AttributesUses
src="location of image"Specifies URL or location of image
height="N"Specifies height of image in pixels
width=N""Specifies width of image in pixels
border="N"Specifies image border in pixels
vspace=N""Specifies vertical space around the image in pixels
hspace="N"Specifies horizontal space around the image in pixels
align="top/bottom/middle/left/right"Aligns the image
title="comments"Specifies comments on the image
alt="alternate text"Specifies alternative text to display while image is not found

Creating Hyperlink

A webpage can contain various links that take you directly to other pages and even specific parts of a given page. These links are known as hyperlinks.Hyperlinks allow visitors to navigate between Web sites by clicking on words, phrases, and images. Thus you can create hyperlinks using text or images available on a webpage.

A anchor tag <a> defines the hyperlink in te HTML and anything between the opening tag and the closing tag becomes part of the link and a user can click that part to reach to the linked document. The major attributes of this tag are href which specifies the link to te next page, target which specifies the space or area where the page will disply, name, etc.
